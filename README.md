# 🚦 ASM-Based Traffic Signal Controller

A digital design project implementing a traffic signal control system using Algorithmic State Machine (ASM) methodology and Finite State Machine (FSM) approach.

## 🎯 Project Overview

This project demonstrates the design and simulation of a robust traffic signal controller for a two-road intersection using ASM-based FSM methodology. The controller manages traffic flow through four well-defined states with precise timing intervals.
